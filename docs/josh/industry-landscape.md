## The DoD IT Channel Overview

Josh operates in a specialized corner of IT sales: **selling technology products and services through the defense/ federal channel to U.S. military branches.**

### Key Characteristics

| Factor | Reality |
|---|---|
| Sales cycle | 12-24+ months typical for enterprise DoD deals |
| Entry barriers | High — security clearances, contract vehicles, incumbent relationships |
| Decision-makers | Multiple layers: end-user → program office → contracting → command |
| Relationship weight | Extremely high — trust is the only currency that moves the needle |
| Contract vehicles | GSA, SEWP, NASA SEWP, IT-70, various branch-specific IDIQs |
| Partner ecosystem | Distributors (like the one Josh came from), VARs, SI's, OEMs |
| Typical deal size | Varies wildly — pilot ($50K) to program ($50M+) |
