## How We Switch Balls

1. **"New ball, new day"** — switching primary mid-day is expensive. Finish the current ball's cycle first.
2. **Voice drops kill spin** — use the Shortcut to log ideas so they don't pull focus mid-task.
3. **Three-bag rule** — never more than 3 spinning balls. If a 4th appears, one must go to holding pattern or die.
4. **Herms calls it** — if I see us adding a 5th ball without retiring one, I'll flag it.
